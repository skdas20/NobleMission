IMAGE GUIDELINES:
-----------------

✓ Use high-quality, professional photos
✓ Ensure all images show diversity and inclusion
✓ Compress images for web (use TinyPNG or similar)
✓ Recommended format: JPG for photos, PNG for graphics
✓ Keep file sizes under 500KB per image
✓ Use consistent color tones across all images
✓ Ensure images are copyright-free or properly licensed

HOW TO REPLACE IMAGES:
----------------------

1. Place your images in this folder (images/)
2. Name them according to the list above
3. Open noble-mission.html in a text editor
4. Replace the image URLs:

   FROM: src="https://images.unsplash.com/photo-..."
   TO:   src="images/your-image-name.jpg"

Example:
   Line 76: Change
   <img src="https://images.unsplash.com/photo-1503454537195-1dcabb73ffb9?w=1920&h=600&fit=crop"
   TO:
   <img src="images/hero-bg.jpg"
